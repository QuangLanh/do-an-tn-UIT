# Grocery Store Management System

Hệ thống quản lý tạp hóa với Backend (NestJS) và Frontend (React + Vite).

## Cấu trúc dự án

- `back_end_project/` - Backend API (NestJS)
- `front_end_project/` - Frontend Web App (React + Vite)

## Cài đặt

### Backend
cd back_end_project
yarn install
yarn start:dev

### Frontend
cd front_end_project
yarn install
yarn dev
